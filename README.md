# Network Programming Report

This project is a **UAV/Drone control and monitoring server** using MAVLink protocol, built with Python + Flask for **simulating and controlling USV on PX4 platform**.

Developed by Lê Thanh Trà - 106220273, Trần Đại Tín - 106220272 và Nguyễn Xuân Trường - 106220275 lớp 22KTMT2

---

## Repository Structure
```
Network_Programing_Report/
├── src/          # Source code
└── docs/         # Project report
```

### Directories

- **`src/`**: Contains all source code for the server application. See [src/README.md](src/README.md) for detailed documentation.
- **`docs/`**: Contains project reports.
